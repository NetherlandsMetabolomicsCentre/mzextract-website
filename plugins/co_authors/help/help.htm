Kurz-Dokumentation Plugin "CoAuthors_XH"

Version CoAuthors_XH v1.0 - released 06/2012

CoAuthors_XH ist ein CMSimple_XH Plugin, mit dem txt-Dateien, die html Quelltext enthalten und die irgendwo in einem Unterverzeichnis einer CMSimple_XH Installation liegen, in eine Seite dieser CMSimple_XH Installation eingebunden werden können.

Das Plugin entfernt alle h1 Überschriften aus dem Quelltext.

Mehr Informationen gibt es hier:   CMSimpleCoAuthors Projektseite »

1. Systemvoraussetzungen

Voraussetzung ist eine funktionierende Installation von CMSimple_XH 1.2 utf-8 oder höher.

Das Plugin "CoAuthors_XH" wird nur utf-8 codiert bereitgestellt.

2. Installation

  1. zip-Datei herunterladen und mit einem geeigneten Programm entpacken
  2. Laden Sie Ordner und Dateien des Verzeichnisses "plugins" in das Verzeichnis "plugins" Ihrer CMSimple_XH Installation hoch.

3. Datei- und Ordnerrechte

Die Vergabe von Schreibrechten für Ordner und Dateien ist nicht notwendig.

4. Plugin-Aufruf

Fügen Sie folgenden Plugin-Aufruf in Ihre CMSimple_XH Seite ein, an der Stelle, an der die txt-Datei eingebunden werden soll:

{{{PLUGIN:co_authors('CoAuthorFolder/','Die_Autoren_Seite');}}}

Da die Editoren gerne automatich ein <p> ... </p> um den Plugin Aufruf setzen, empfiehlt es sich, den Plugin Aufruf in ein <div> ... </div> zu packen, damit die Seite validiert.

Hinweis: Bitte fügen Sie den Plugin Aufruf immer in der Quelltext Ansicht ein.

CoAuthorFolder: Das Verzeichnis, in dem CMSimple_XH CoAuthors - also die Autoren Installation - installiert ist.

!!! WICHTIG !!! der Slash ("/") am Ende, ohne den funktioniert's nicht.

Die_Autoren_Seite: Der Name der Seite der Autoren Installation, die eingebunden werden soll, mit "_" für die Leerstellen.

!!! WICHTIG !!! Bitte bei den Namen der Autorenseiten keine Umlaute oder Sonderzeichen verwenden, da daraus Dateinamen gemacht werden, die an CMSimple_XH übergeben werden.

5. Konfiguration

Für CoAuthors_XH gibt es keine Konfigurations Optionen.
